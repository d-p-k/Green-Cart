This is a Grocery Delivery project using React & Spring Boot

-->> Initial Commands
1. bun create vite@latest . (if you don't type this "." (dot), it'll create folder in your current folder and it'll install all the files there)
    - give project name
    - select framework (choose react here)
    - select variant (choose javascript here)
2. bun i (it'll create/update node_modules folder with all required pkgs)
3. to remove prop-types error, include this line ("react/prop-types": "off",) into rules in eslint.confing.js file.
4. include this line too ("react/no-unescaped-entities": "off",) into rules in eslint.confing.js file because when you type "It's" it gives error because of use of '.
5. bun run dev 

-->> Install Tailwind CSS
go to this website do accordingly -> https://tailwindcss.com/docs/installation/using-vite
in case if any doubt, watch this (8 min YT video of installation of TailwindCSS in React) -> https://youtu.be/5yPX6HYelqw?si=lvIb1croEYyuxxYK

--> Most Important Step
For auto suggestion of tailwindcss properties,
go to settings -> search "Tailwind CSS Experimental Config File" -> then open settings.json file
find this property - "tailwindCSS.experimental.configFile"
then add relative path of index.css file in-place of null. See example below.
"tailwindCSS.experimental.configFile": "src/index.css", 
if above line doesn't work then use relative path like below line:
"tailwindCSS.experimental.configFile": "GreenCart/src/index.css",

-->> Install Some Extra Pkgs
bun i react-router-dom (to navigate different pages)
bun i react-hot-toast (to show toast notifications)
bun i axios (to call backend apis)

-->> _redirects File
This file is required when you deploy your frontend on Netlify. 
Make this file inside public folder and then build. Then drag & drop the dist folder on Netlify, your project will be live in seconds.
If you don't include this file and build your project. In that case if you try manually typing this url (https://greencartshop.netlify.app/seller) and hit, you'll see page not found issue.
